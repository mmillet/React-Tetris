![React Tetris](http://120.25.62.254:90/tetris.gif "React Tetris")

在这里可以玩: http://120.25.62.254:8080/

## Get start

1.  npm install fis3 -g
2.  npm install
3.  fis3 release (打包: fis3 release prod)
4.  fis3 server start
